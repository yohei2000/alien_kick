# エイリアンキックバスター

スマホで遊べる、リズムアクション型のPKバトルゲームです。プレイヤーはキッカーになり、音楽に合わせて飛んでくるボールをボレーシュートします。ハードヒットでコンボをつなぎ、エイリアンGKを全員倒すと勝利です。

## ゲーム内容

- エイリアンがGK、プレイヤーがキッカー
- BPM 126の音楽に合わせたリズムパターンでボールがフィードされる
- タップのタイミングが良いほどハードヒットになり、ゴール率とダメージが上がる
- コンボが伸びると音楽とキック威力が段階的に強くなる
- Slime / Mantis / Psychic の3体をすべて倒すとクリア
- キッカーとGKはPhaser上で状態アニメーションと画像アセットを組み合わせて描画

## 同期と音楽

Web Audioで生成したBGMを基準に、ボールの到達タイミング、リズムガイド、タップ判定を同期しています。実際に聞こえるタイミングに合わせるため、AudioContextの出力遅延を見た目と判定に反映しています。

音楽はドラム、和音、リード、敵ごとの音色、リズムガイドに分けてミックスしています。キック、ゴール、ブロック、ダウン時にはBGM側を一瞬引っ込め、シュート音とエフェクトが前に出るようにしています。

## ローカル起動

```powershell
npm.cmd run dev
```

ブラウザで `http://localhost:4173` を開いてください。
